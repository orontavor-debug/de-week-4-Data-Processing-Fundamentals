# Web Scraping https://www.imdb.com/chart/top/

## Objective:

Scrape the titles and ratings of the top 25 movies from IMDB Top 250 Movies using BeautifulSoup (bs4).

## Instructions:

1. Access IMDB's Top 250 https://www.imdb.com/chart/top/ website to scrap information of first 25 movies, not all 250.
2. Store results in a Pandas DataFrame with two columns: Title and Rating.
3. Display the top 10 results as a preview.

4. Save the DataFrame as a CSV file top25_movies.csv.
